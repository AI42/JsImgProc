Name: JsImageProc
A little image proceesing software, developed by Zhu, Jiahao(jethro.zjh@gmail.com)
using C++, OpenCV, QT.

	-----------------------------------------|
	View Layer:  GUI -- designed using QT    |
	-----------------------------------------|
	Controller Layer:        C++             | 
	-----------------------------------------|
	Module Layer:       C++ & OpenCV         |
	-----------------------------------------|

Features:
	1.SISR
	2.Common Image Enhancement features such as blur, edge detect....

There are 4 modules in this proj:
	1.dbgsupport
  		--> some debug support macros and functions
	2.mods_and_ctrls
  		--> functional classes in module layer and controller layer
	3.ocv_SISR
		--> SISR algorithm implemented in imperative style
	4.J_ImgprocVS
		--> a QT GUI proj, imported to VS


Build Enviroment:
	1. OpenCV 2.4.5
	2. QT 5.0
	3. Visual Studio 2012 ultimate
